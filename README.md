# CSV Docs Converter

## Docker
#### Simply run container 
```docker-compose up```
#### WEB should be available at http://localhost:3099/

## Usage no-docker

#### 1) Install dependencies 
```npm install```


#### 2) Put files to process in [files-input](files-input)

#### 3) Run the script 
```node index.js```

#### 4) Processed files will appear in  [files-output](files-output)
